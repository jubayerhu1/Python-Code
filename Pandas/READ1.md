# What is Pandas?
<P> Pandas is Python library used for working with data sets.
It has functions for <b>analyzing , cleaning , exploring and manipulating data</b>
The name "Pandas" has a reference to both "Panel Data", and "Python Data Analysis" and was created by Wes Mckinney in 2008.</p>

## Why use Pandas?

<p> Pandas allows us to analyze <u>big data and make conclusion based on statistical theories</u>.
Pandas can clean messy data sets and make them readable and relevant.
Relevant data is very important in data science.</p>


## What can Pandas Do ?

<p> Pandas gives you answers about the data like:
- is there a correlation between two or more columns?
- What is average value?
- Max value?
- Min value?
Pandas are also able to detete rows that are not relevant, or contains wrong value, like empty or NULL values. This is called <i>cleaning</i> data.
